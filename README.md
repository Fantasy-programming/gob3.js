# Gob3.js

A Simple library for Building UIs

- Server side rendering
- Backend Agnostic
- Lightweight
- Multiple page / SPA (with ssr)
- As simple as htmx
- Signals (like events but better)
- Components
- Zustand like state management
- Solid & Vue like reactivity
- State of the art DX
- uses $ as a prefix (like jQuery)
